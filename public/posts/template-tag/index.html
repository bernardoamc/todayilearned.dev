template tag

  • html
  • javascript
  • security

TIL about the template tag. It’s used to hold client-side content that is not to be rendered when a page is loaded but may subsequently be instantiated during runtime using JavaScript.

Let’s see some client side input without the template tag:

  let container = document.createElement('div');

  // This will trigger the alert
  container.innerHTML = "<img src=x onerror=alert(1)>"

Now let’s use the <template> tag:

  let container = document.createElement('template');

  // Alert NOT triggered
  container.innerHTML = "<img src=x onerror=alert(1)>"

We can access the content inside a template with template.content and change the rendered HTML before actually inserting it on the page. There’s a great video from LiveOverflow explaining how this tag is used by Google and many others in order to prevent XSS client-side and how a misconfiguration allowed XSS on Google Search.
