will change

  • css

TIL about the will-change CSS property. It provides hints to browsers about how an element is expected to change. Browsers may set up optimizations before an element is actually changed.

.sidebar {
  will-change: transform;
}

*This property should be used as a last resort, try to optimize before relying on it.

For example, avoid transitions with top/right/bottom/left properties as they force the browser to create layouts every time.
